These files are shortened versions of the initial data driven test and verify JSON files. They are intended to provide a small starting set for testing and checking the Conformance work on GitHub and on user systems.


The 3 test files contain data that can be processed by text executors, supplied by the test driver.


The 3 verify files have expected results for these test cases (and more, in some files). These can be used to check that the verifier and test reporting work correctly.
